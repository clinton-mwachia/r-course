Introduction

Logistic regression is a widely used algorithm for binary classification problems in machine learning. In this lesson, we will explore how to implement machine learning using logistic regression in R. We will cover the entire process, from loading and cleaning the data to model training, evaluation, and making predictions.

The Data

The sinking of the Titanic is one of the most infamous shipwrecks in history.

On April 15, 1912, during her maiden voyage, the widely considered “unsinkable” RMS Titanic sank after colliding with an iceberg. Unfortunately, there weren’t enough lifeboats for everyone onboard, resulting in the death of 1502 out of 2224 passengers and crew.

While there was some element of luck involved in surviving, it seems some groups of people were more likely to survive than others.

In this challenge, we will build a predictive model that answers the question: “what sorts of people were more likely to survive?” using passenger data (ie name, age, gender, socio-economic class, etc). We will use the titanic dataset in this lesson, you can learn more from the kaggle website.

Modelling

Splitting the Data

We will split the data into: -

  1. Training Data (80%).
  2. Validation Data (20%).
  
# splitting index
index = createDataPartition(imputed_data$Survived, p=0.8, list = FALSE)

# training data
training = imputed_data[index,]

# validation data
validation = imputed_data[-index,]

Fitting Base model

# the model
model = glm(Survived~., data = training, family = binomial(link="logit"))

# model summary

Visualize results

To plot the results of a logistic regression model, you can create various types of visualizations depending on the specific requirements and objectives. Here are a few common ways to visualize the results of a logistic regression model in R:

  **ROC Curve**
  
The Receiver Operating Characteristic (ROC) curve is a widely used graphical          representation to evaluate the performance of a binary classifier. It shows the       trade-off between the true positive rate (sensitivity) and the false positive         rate (1 - specificity) at various classification thresholds.
# ROC curve
library(pROC)
## Type 'citation("pROC")' for a citation.
## 
## Attaching package: 'pROC'
## The following objects are masked from 'package:stats':
## 
##     cov, smooth, var
roc_curve <- roc(validation$Survived, pred)
## Setting levels: control = 0, case = 1
## Setting direction: controls < cases
plot(roc_curve, main = "ROC Curve", col = "blue")

**Coefficient Plot**
  
 A coefficient plot can be used to visualize the estimated coefficients of the        logistic regression model, providing insights into the importance and direction of     each variable's impact on the outcome.`
coef_data <- data.frame(variable = names(coef(model)), coefficient = coef(model))

ggplot(coef_data, aes(x = variable, y = coefficient)) + 
  geom_hline(yintercept = 0, linetype = "dashed") + 
  geom_bar(stat = "identity", fill = "blue") + 
  coord_flip() +
  labs(title = "Logistic Regression Coefficient Plot") +
  theme_bw()
